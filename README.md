# Iris Flower Classification 

A machine learning project that classifies Iris flowers into three species
(Setosa, Versicolor, Virginica) based on petal and sepal measurements.


This is my first machine learning project, built while learning ML concepts
during my first year of B.Tech Civil Engineering at MANIT Bhopal(2025-2029).

What I've used:
- Python
- scikit-learn
- pandas
- seaborn
- matplotlib

The project:
- Loads and explores the Iris dataset
- Visualizes feature relationships using pairplots
- Trains a Logistic Regression model
- Evaluates accuracy using confusion matrix and classification report
- Compares Logistic Regression vs Naive Bayes performance


Achieved 100% accuracy on the test set using Logistic Regression.

## 🚀 How to Run
1. Clone this repository
2. Install dependencies: `pip install pandas scikit-learn seaborn matplotlib`
3. Run `Iris classification model.py`
